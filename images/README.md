# images フォルダ

ここに下記のファイル名（すべて **WEBP形式**）で画像を保存すると、LPの該当箇所に自動で表示されます。
ファイル名はLP側と一致させてください（大文字・小文字も同じに）。画像が未設置のスロットは、
ファイル名とサイズが書かれたプレースホルダが表示されます。

※ファーストビューのメイン画像は「動くアプリデモ」に変更したため不要です。
※コイン装飾はCSSで描画しているため画像は不要です。

### スマホ専用フルFV画像（必須・スマホ表示で使用）

スマホ表示では、この1枚絵をFVとして全幅表示します（PC表示では使いません）。

| ファイル名 | サイズ(px) | 内容 |
|---|---|---|
| `fv-sp.webp` | 887×1774 | スマホ用のFV全体（見出し・アプリ画面・キャラ・コイン・機能などを1枚に） |

※注意書き（※本アプリは〜／※暗号資産の取引には〜）は画像には入れず、画像の下に実テキストで表示しています。

### ファーストビューの背景キャラクター（PC用・任意）

共有画風（明るいフレンドリーなイラスト）で、**背景が透過**の立ち姿を作ってください。
FV下部の左右コーナーに立つ配置です。**顔・体は内側（中央）を向く**ように。

| ファイル名 | サイズ(px) | 内容 |
|---|---|---|
| `char-left.webp` | 520×760・**透過** | 左下に立つ人物（例：やわらかい雰囲気の女性）。右（中央）を向く |
| `char-right.webp` | 520×760・**透過** | 右下に立つ人物（例：ノートPCを持つ男性）。左（中央）を向く |

## 作成のコツ

- ページの配色に合わせ、**白背景＋ブルー(#1B6FD0)＋ゴールド(#E5B94E)** のトーンで揃えると統一感が出ます。
- `app-ui-*` の3枚は、実物のアプリのスクリーンショットがあればそちらの方が信頼感が出ます。
