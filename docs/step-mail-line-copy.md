# ステップメール用｜LINE誘導文（コピー集）

ステップメールやメルマガ末尾に入れる、LINE友だち追加への誘導文のストックです。
LINE友だち追加URL（`https://lin.ee/nxAgf9Z`）は反映済み。そのまま使えます。

> ポイント：LINEに来てもらう「理由（メリット）」を添えると登録率が上がります。

---

## 案A：おすすめ（丁寧・メリット提示）

```
▼LINEでも受け取れます
資料の続報や、注意が必要な銘柄のお知らせは、LINEでも配信しています。
メールより手軽に受け取りたい方は、こちらからご登録ください。
▶ https://lin.ee/nxAgf9Z
```

## 案B：短く（1〜2行）

```
📱 LINEでも配信中
最新の分析情報を、LINEでも受け取れます。ご登録はこちら → https://lin.ee/nxAgf9Z
```

## 案C：やわらかい語り口

```
Crypto Novaは、LINEでも情報をお届けしています。
「メールだと見逃しがち」という方は、ぜひLINEでもつながってください。
▶ LINE友だち追加はこちら：https://lin.ee/nxAgf9Z
```

## 案D：フッター向け（各メール末尾に常設）

```
─────────────
● LINEでも配信中｜見逃したくない方はこちら → https://lin.ee/nxAgf9Z
─────────────
```

---

## 使い方のコツ

- **1通目（登録直後）**は案A（メリットを丁寧に）。
- **2通目以降**は案D（短い1行）で十分。
- ボタンにできるなら、リンク文言は **「LINEで友だち追加」** が最もクリックされやすい。

## 注意（表現）

- 「儲かる」「急騰」などの**煽り表現は入れない**こと。
- LINE配信も特定電子メール法・LINE社の規約の対象。上記の案はすべて「情報を受け取れる」中立表現。
