# A/Bテスト計画｜FVキャッチコピー（リスト獲得改善）

## 背景
- 現行LP（index.html）は資料請求（リスト取り）がほぼゼロ
- 原因仮説：①見出しが抽象的で初見に伝わらない ②CTAが少ない（ナビ＋最下部のみ）③特典（登録の理由）がない
- index2/index3 でこの3点を改善し、キャッチコピー2案を比較する

## バリエーション

| ページ | キャッチ | 見出し |
|---|---|---|
| **index2.html**（案2） | "次のビットコイン"を夢見る。その前に、データを見る。 | ビットコインの成長曲線に"似た銘柄"を、AIが探して30個並べるアプリ。 |
| **index3.html**（案3） | 夢を見たっていい。ただし、噂ではなくデータで。 | （同上） |

- 見出し・本文・CTA・特典はすべて共通。**違いはキャッチ1行のみ**（純粋な比較のため）
- 両ページとも noindex（検索に載らない）
- テストページはスマホもテキストFV表示（fv-sp画像は使わない。キャッチの差を出すため）
- index.html（現行）は無変更のまま温存

## 共通の改善点（両バリエーションに搭載）
1. CTAを6か所に増設（FV直下／仕組み後／できること後／特典／最終／スマホ固定バー）
2. 資料請求特典セクション新設：
   - 特典①：**【保存版】はじめての暗号資産ガイド**
   - 特典②：**【保存版】世界の暗号資産にアクセスする手順書**
   - ※特典はメール／LINEでお届け。利益保証をしない旨の注意書きあり
3. 太字の再設計（類似率順30銘柄を最強調）

## 配信方法（SmartNews想定）
- 同一キャンペーン内に広告を2本作り、リンク先だけ変える
  - 広告A → https://cryptonova.jp/sn/index2.html
  - 広告B → https://cryptonova.jp/sn/index3.html
- 予算は均等配分。クリエイティブ（バナー・広告文）は共通にする
- 新オーディエンス案：年齢45〜65／興味関心＝経済・金融／iOS厚め（小金持ち近似）

## 判定
- 主要指標：**資料請求数（UTAGEフォーム到達→登録）**
- 補助指標：LPクリック率、スクロール率
- 期間目安：1〜2週間 or 各100クリック以上たまってから判断
- 勝った方を index.html に昇格（クライアント承認を取ってから）
